Asymmetric Adaptive Heterogeneous Network for Multi-Modality Medical Image Segmentation
еҸ‘иЎЁдәҺпјҡ2025-07-05 | еҲҶзұ»пјҡ еӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸеҲҶеүІ
еӯ—ж•°з»ҹи®Ў: 3.5k | йҳ…иҜ»ж—¶й•ҝ: 12еҲҶй’ҹ | йҳ…иҜ»йҮҸпјҡ

йҮҚеәҶйӮ®з”өеӨ§еӯҰгҖҒз¬¬дёүеҶӣеҢ»еӨ§еӯҰгҖҒйҮҚеәҶеҢ»з§‘еӨ§еӯҰз¬¬дәҢйҷ„еұһеҢ»йҷў

ж‘ҳиҰҒ

Existing studies of multi-modality medical image segmentation tend to aggregate all modalities without discrimination and employ multiple symmetric encoders or decoders for feature extraction and fusion. They often over-look the different contributions to visual representation and intelligent decisions among multi-modality images. Motivated by this discovery, this paper proposes an asymmetric adaptive heterogeneous network for multi-modality image feature extraction with modality discrimination and adaptive fusion. For feature extraction, it uses a heterogeneous two-stream asymmetric feature-bridging network to extract complementary features from auxiliary multi-modality and leading single-modality images, respectively. For feature adaptive fusion, the proposed Transformer-CNN Feature Alignment and Fusion (T-CFAF) module enhances the leading single-modality information, and the Cross-Modality Heterogeneous Graph Fusion (CMHGF) module further fuses multi-modality features at a high-level semantic layer adaptively. Comparative evaluation with ten segmentation models on six datasets demonstrates significant efficiency gains as well as highly competitive segmentation accuracy. ((Our code is publicly available at https://github.com/joker-527/AAHN)

зҝ»иҜ‘

зҺ°жңүзҡ„еӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸеҲҶеүІз ”з©¶еҫҖеҫҖеңЁдёҚеҠ еҢәеҲҶең°иҒҡеҗҲжүҖжңүжЁЎжҖҒпјҢе№¶йҮҮз”ЁеӨҡдёӘеҜ№з§°зј–з ҒеҷЁжҲ–и§Јз ҒеҷЁиҝӣиЎҢзү№еҫҒжҸҗеҸ–е’ҢиһҚеҗҲж—¶пјҢеҝҪи§ҶдәҶеӨҡжЁЎжҖҒеӣҫеғҸеңЁи§Ҷи§үиЎЁзӨәе’ҢжҷәиғҪеҶізӯ–дёӯзҡ„дёҚеҗҢиҙЎзҢ®гҖӮеҸ—жӯӨеҸ‘зҺ°зҡ„еҗҜеҸ‘пјҢжң¬ж–ҮжҸҗеҮәдәҶдёҖз§Қз”ЁдәҺеӨҡжЁЎжҖҒеӣҫеғҸзү№еҫҒжҸҗеҸ–зҡ„йқһеҜ№з§°иҮӘйҖӮеә”ејӮжһ„зҪ‘з»ңпјҢиҜҘзҪ‘з»ңе…·жңүжЁЎжҖҒеҢәеҲҶе’ҢиҮӘйҖӮеә”иһҚеҗҲеҠҹиғҪгҖӮеңЁзү№еҫҒжҸҗеҸ–ж–№йқўпјҢдҪҝз”ЁејӮжһ„еҸҢжөҒйқһеҜ№з§°зү№еҫҒжЎҘжҺҘзҪ‘з»ңеҲҶеҲ«д»Һиҫ…еҠ©еӨҡжЁЎжҖҒе’Ңдё»иҰҒеҚ•жЁЎжҖҒеӣҫеғҸдёӯжҸҗеҸ–дә’иЎҘзү№еҫҒгҖӮеҜ№дәҺзү№еҫҒиҮӘйҖӮеә”иһҚеҗҲпјҢжҸҗеҮәзҡ„Transformer-CNNзү№еҫҒеҜ№йҪҗдёҺиһҚеҗҲпјҲT-CFAFпјүжЁЎеқ—еўһејәдәҶдё»иҰҒеҚ•жЁЎжҖҒдҝЎжҒҜпјҢиҖҢи·ЁжЁЎжҖҒејӮжһ„еӣҫиһҚеҗҲпјҲCMHGFпјүжЁЎеқ—иҝӣдёҖжӯҘеңЁй«ҳзә§иҜӯд№үеұӮиҮӘйҖӮеә”ең°иһҚеҗҲеӨҡжЁЎжҖҒзү№еҫҒгҖӮдёҺе…ӯдёӘж•°жҚ®йӣҶдёҠзҡ„еҚҒдёӘеҲҶеүІжЁЎеһӢиҝӣиЎҢжҜ”иҫғиҜ„дј°пјҢз»“жһңжҳҫзӨәжҳҫи‘—зҡ„ж•ҲзҺҮжҸҗеҚҮд»ҘеҸҠжһҒе…·з«һдәүеҠӣзҡ„еҲҶеүІеҮҶзЎ®жҖ§гҖӮпјҲжҲ‘д»¬зҡ„д»Јз ҒеңЁhttps://github.com/joker-527/AAHNе…¬ејҖжҸҗдҫӣпјү

з ”з©¶иғҢжҷҜ

еӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸиғҪд»ҺеӨҡдёӘи§’еәҰжҸҗдҫӣдә’иЎҘдҝЎжҒҜпјҢж•ҙеҗҲдёҚеҗҢжҲҗеғҸжЁЎжҖҒеҸҜжҸҗеҚҮжҲҗеғҸж•°жҚ®зҡ„ж•Ҹж„ҹжҖ§гҖҒзү№ејӮжҖ§е’Ңз©әй—ҙеҲҶиҫЁзҺҮгҖӮз„¶иҖҢпјҢзҺ°жңүеӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸеҲҶеүІз ”з©¶еӯҳеңЁдёӨдёӘе…ій”®й—®йўҳпјҡ

  1. зү№еҫҒжҸҗеҸ–й—®йўҳпјҡзҺ°жңүж–№жі•еҖҫеҗ‘дәҺж— е·®еҲ«иҒҡеҗҲжүҖжңүжЁЎжҖҒпјҢйҮҮз”ЁеҜ№з§°зј–з ҒеҷЁжҲ–и§Јз ҒеҷЁиҝӣиЎҢзү№еҫҒжҸҗеҸ–пјҢеҝҪз•ҘдәҶдёҚеҗҢжЁЎжҖҒеӣҫеғҸеҜ№и§Ҷи§үиЎЁзӨәе’ҢжҷәиғҪеҶізӯ–зҡ„дёҚеҗҢиҙЎзҢ®пјҢйҡҫд»Ҙжңүж•ҲжҸҗеҸ–еӨҡжЁЎжҖҒеӣҫеғҸдёӯзӢ¬зү№дё”дә’иЎҘзҡ„зү№еҫҒгҖӮ
  2. зү№еҫҒиһҚеҗҲй—®йўҳпјҡеёёи§Ғзҡ„зү№еҫҒиһҚеҗҲж–№жі•жҳҜеңЁйҖҡйҒ“з»ҙеәҰдёҠиҝӣиЎҢеҲқе§ӢжӢјжҺҘжҲ–жұӮе’ҢпјҢз„¶еҗҺиҫ“е…Ҙзү№е®ҡиһҚеҗҲжЁЎеқ—пјҢиҝҷз§Қж–№ејҸеҸҜиғҪеҜјиҮҙжҖ§иғҪз“¶йўҲпјҢж— жі•е®һзҺ°еӨҡжЁЎжҖҒзү№еҫҒзҡ„иҮӘйҖӮеә”иһҚеҗҲпјҢиҝҳеҸҜиғҪйҖ жҲҗзӢ¬зү№зү№еҫҒзҡ„дёўеӨұгҖӮ

дёәи§ЈеҶіиҝҷдәӣй—®йўҳпјҢжң¬ж–ҮжҸҗеҮәдәҶдёҖз§Қз”ЁдәҺеӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸзү№еҫҒжҸҗеҸ–зҡ„йқһеҜ№з§°иҮӘйҖӮеә”ејӮжһ„зҪ‘з»ңпјҢж—ЁеңЁе®һзҺ°жЁЎжҖҒеҢәеҲҶе’ҢиҮӘйҖӮеә”иһҚеҗҲпјҢд»Ҙжңүж•Ҳз®ЎзҗҶеӨҡжЁЎжҖҒзү№еҫҒзҡ„ејӮиҙЁжҖ§пјҢжҳҫи‘—жҸҗй«ҳеҲҶеүІз»“жһңгҖӮ

з ”з©¶зҺ°зҠ¶

  • еӨҡжЁЎжҖҒзү№еҫҒжҸҗеҸ–пјҡеёёи§Ғж–№жі•жңүйӣҶдёӯжҸҗеҸ–пјҲеҰӮ3D AGSE - VNetгҖҒMBANetгҖҒSwin UNETRпјүе’ҢеҚ•зӢ¬жҸҗеҸ–пјҲеҰӮME - NetгҖҒMAMLгҖҒNestedFormerгҖҒA2FSegпјүпјҢдҪҶеӨ§еӨҡеҹәдәҺеҜ№з§°зү№еҫҒжҸҗеҸ–пјҢжңӘиҖғиҷ‘жЁЎжҖҒе…ізі»гҖӮ
  • зү№еҫҒиһҚеҗҲпјҡзҺ°жңүж–№жі•еёёйҮҮз”ЁеҲқе§ӢжӢјжҺҘжҲ–жұӮе’ҢпјҢеҶҚйҖҡиҝҮзү№е®ҡиһҚеҗҲжЁЎеқ—еӨ„зҗҶпјҢеҰӮIVD - NetгҖҒWangзӯүдәәе’ҢZhouзӯүдәәзҡ„ж–№жі•пјҢдҪҶеҸҜиғҪеҜјиҮҙжҖ§иғҪз“¶йўҲпјҢдё”з®ҖеҚ•дҪҝз”ЁCNNжҲ–TransformerдјҡжҚҹеӨұзӢ¬зү№зү№еҫҒгҖӮ

  • еӣҫзҘһз»ҸзҪ‘з»ңеә”з”Ёпјҡз”ЁдәҺеҲҶеүІзҡ„еӣҫз»“жһ„жЁЎеһӢдё»иҰҒе…іжіЁеҚ•жЁЎжҖҒпјҢйҡҫд»ҘеӨ„зҗҶеӨҡжЁЎжҖҒеӣҫеғҸзҡ„ејӮиҙЁжҖ§гҖӮ

жҸҗеҮәзҡ„жЁЎеһӢ

ж•ҙдҪ“жһ¶жһ„

жЁЎеһӢзҡ„ж•ҙдҪ“жһ¶жһ„еҢ…жӢ¬дёҖдёӘйқһеҜ№з§°еҸҢеҲҶж”Ҝзү№еҫҒжЎҘжҺҘзј–з ҒеҷЁгҖҒTransformer - CNN зү№еҫҒеҜ№йҪҗдёҺиһҚеҗҲпјҲT - CFAFпјүжЁЎеқ—гҖҒи·ЁжЁЎжҖҒејӮиҙЁеӣҫиһҚеҗҲпјҲCMHGFпјүжЁЎеқ—е’Ңи§Јз ҒеҷЁпјҢдё»иҰҒжөҒзЁӢдёәпјҡ

  1. зү№еҫҒжҸҗеҸ–пјҡдҪҝз”ЁйқһеҜ№з§°еҸҢеҲҶж”Ҝзү№еҫҒжЎҘжҺҘзј–з ҒеҷЁд»Һиҫ…еҠ©еӨҡжЁЎжҖҒе’Ңдё»еҜјеҚ•жЁЎжҖҒеӣҫеғҸдёӯеҲҶеҲ«жҸҗеҸ–дә’иЎҘзү№еҫҒгҖӮ
  2. зү№еҫҒиһҚеҗҲпјҡT - CFAF жЁЎеқ—еўһејәдё»еҜјеҚ•жЁЎжҖҒдҝЎжҒҜпјҢCMHGF жЁЎеқ—еңЁй«ҳеұӮиҜӯд№үеұӮиҮӘйҖӮеә”ең°иһҚеҗҲеӨҡжЁЎжҖҒзү№еҫҒгҖӮ
  3. еӣҫеғҸеҲҶеүІпјҡи§Јз ҒеҷЁйҖҡиҝҮи·іи·ғиҝһжҺҘйҖҗжӯҘеҜ№йҪҗжқҘиҮӘ CNN еҲҶж”Ҝзҡ„дҪҺеұӮж¬Ўе’Ңй«ҳеұӮж¬Ўзү№еҫҒпјҢе®һзҺ°жңҖз»Ҳзҡ„еҲҶеүІгҖӮ

дё»иҰҒз»„д»¶

  • йқһеҜ№з§°еҸҢеҲҶж”Ҝзү№еҫҒжЎҘжҺҘзј–з ҒеҷЁпјҡ
    • еӨҡжЁЎжҖҒ Transformer еҲҶж”Ҝпјҡе…·жңүејәеӨ§зҡ„е…ЁеұҖдёҠдёӢж–ҮдҝЎжҒҜе»әжЁЎиғҪеҠӣпјҢиғҪеӨҹжӣҙеҘҪең°жҚ•жҚүиҫ…еҠ©еӨҡжЁЎжҖҒзү№еҫҒгҖӮиҜҘеҲҶж”ҜеҢ…жӢ¬еөҢе…ҘеұӮе’ҢйӘЁе№Із»“жһ„гҖӮеөҢе…ҘеұӮйҮҮз”Ёдј з»ҹеҚ·з§Ҝз»“жһ„пјҢеҸҜдҝқз•ҷдҪҚзҪ®дҝЎжҒҜе№¶жҸҗеҸ–иҜҰз»Ҷзү№еҫҒпјӣйӘЁе№Із»“жһ„жҳҜ CNN е’Ң ViT зҡ„ж··еҗҲз»„еҗҲпјҢз»“еҗҲдәҶдёӨиҖ…зҡ„дјҳеҠҝпјҢиғҪеӨҹдҝқз•ҷ CNN зҡ„еҮҶзЎ®дҪҚзҪ®дҝЎжҒҜе’Ң Transformers зҡ„е…ЁеұҖдёҠдёӢж–Үе»әжЁЎиғҪеҠӣгҖӮ
    • еҚ•жЁЎжҖҒ CNN еҲҶж”ҜпјҡеҲ©з”Ё CNN дҝқз•ҷеұҖйғЁз»ҶиҠӮзҡ„иғҪеҠӣеӨ„зҗҶдё»еҜјеҚ•жЁЎжҖҒеӣҫеғҸгҖӮдё»иҰҒз”ұеӨҡдёӘж®Ӣе·®еқ—пјҲRESпјүе’Ң T - CFAF жЁЎеқ—з»„жҲҗпјҢйҖҡиҝҮе№іеқҮжұ еҢ–еҪўжҲҗзү№еҫҒйҮ‘еӯ—еЎ”з»“жһ„пјҢз”ЁдәҺжҸҗеҸ–еӨҡе°әеәҰзү№еҫҒгҖӮ
  • Transformer - CNN зү№еҫҒеҜ№йҪҗдёҺиһҚеҗҲпјҲT-CFAFпјүжЁЎеқ—пјҡдёәдәҶеўһејә CNN еҲҶж”ҜдёҠдё»еҜјеҚ•жЁЎжҖҒдҝЎжҒҜпјҢиҜҘжЁЎеқ—е°ҶеӨҡжЁЎжҖҒзү№еҫҒеӣҫе’ҢеҚ•жЁЎжҖҒзү№еҫҒеӣҫеңЁйҖҡйҒ“з»ҙеәҰдёҠжӢјжҺҘпјҢйҖҡиҝҮи®Ўз®—йҖҡйҒ“жіЁж„ҸеҠӣе’Ңи·ЁжЁЎжҖҒжіЁж„ҸеҠӣпјҢи°ғж•ҙйҖҡйҒ“жқғйҮҚпјҢиЎҘе……еӨҡжЁЎжҖҒдҝЎжҒҜпјҢеўһејәдё»еҜј CNN зү№еҫҒгҖӮ
  • и·ЁжЁЎжҖҒејӮжһ„еӣҫиһҚеҗҲпјҲCMHGFпјүжЁЎеқ—пјҡз”ЁдәҺзј“и§ЈдёҚеҗҢжЁЎжҖҒд№Ӣй—ҙзҡ„зү№еҫҒејӮиҙЁжҖ§пјҢе®һзҺ°и·ЁжЁЎжҖҒзү№еҫҒзҡ„иҮӘйҖӮеә”иһҚеҗҲгҖӮиҜҘжЁЎеқ—е°Ҷзү№еҫҒеӣҫз©әй—ҙжҠ•еҪұеҲ°еӣҫз©әй—ҙпјҢйҖҡиҝҮеӣҫеҚ·з§Ҝж“ҚдҪңжӣҙж–°иҠӮзӮ№зү№еҫҒпјҢжһ„е»әејӮиҙЁеӣҫпјҢи®Ўз®—иҠӮзӮ№й—ҙзҡ„зӣёдјјеәҰпјҢжӣҙж–°иҠӮзӮ№зү№еҫҒеҗ‘йҮҸпјҢжңҖз»Ҳе®һзҺ°еӨҡжЁЎжҖҒзү№еҫҒзҡ„иһҚеҗҲгҖӮ

(1) еҸӮж•°ж•°йҮҸе’ҢжЁЎеһӢеӨҚжқӮеәҰпјҡиҜҘж–№жі•зҡ„еҸӮж•°ж•°йҮҸе’Ңи®Ўз®—йҮҸиҫғеӨ§пјҢжҜҸжү№ж¬ЎеӨ§е°Ҹдёә1ж—¶пјҢжңү4.1137дәҝдёӘеҸӮж•°е’Ң941.001 GFLOPsпјҢзӣёжҜ”е…¶д»–ж–№жі•и§„жЁЎжӣҙеӨ§гҖӮиҝҷдёҖй—®йўҳеҜјиҮҙеҸ—йҷҗдәҺGPUеҶ…еӯҳпјҢеҸӘиғҪдҪҝз”ЁдәҢз»ҙзҪ‘з»ңгҖӮе°Ҫз®ЎйҮҮз”Ёзҡ„йқһеҜ№з§°зү№еҫҒзј–з Ғз»“жһ„д»ҘеҸҠT - CFAFе’ҢCMHGFзү№еҫҒеҜ№йҪҗиһҚеҗҲжЁЎеқ—иғҪжңүж•ҲеӯҰд№ еӨҡжЁЎжҖҒдҝЎжҒҜе№¶еҸ–еҫ—дјҳдәҺзҺ°жңүе…ҲиҝӣжҠҖжңҜзҡ„з»“жһңпјҢдҪҶеӨ§йҮҸзҡ„еҸӮж•°йҷҗеҲ¶дәҶжЁЎеһӢеңЁеӨ„зҗҶиғҪеҠӣжҲ–еҶ…еӯҳеҸ—йҷҗзҡ„е®һйҷ…еңәжҷҜдёӯзҡ„еә”з”ЁгҖӮ

(2) еӨ„зҗҶдёҚе®Ңж•ҙжЁЎжҖҒж•°жҚ®иғҪеҠӣпјҡиҜҘж–№жі•дё»иҰҒиҒҡз„ҰдәҺеҜ№йҪҗзҡ„еӨҡжЁЎжҖҒеҢ»еӯҰеӣҫеғҸеҲҶеүІгҖӮз„¶иҖҢеңЁе®һйҷ…дёҙеәҠзҺҜеўғдёӯпјҢз”ұдәҺи®ҫеӨҮйҷҗеҲ¶жҲ–жӮЈиҖ…зҠ¶еҶөзӯүеҺҹеӣ пјҢеҸҜиғҪдјҡеҮәзҺ°жЁЎжҖҒзјәеӨұзҡ„жғ…еҶөгҖӮжңӘжқҘз ”з©¶йңҖиҰҒи§ЈеҶіеңЁжЁЎжҖҒж•°жҚ®дёҚе®Ңж•ҙзҡ„жғ…еҶөдёӢиҝӣиЎҢжңүж•ҲеҲҶеүІзҡ„й—®йўҳпјҢеҚіејҖеҸ‘еҮәиғҪеӨҹйҖӮеә”жЁЎжҖҒзјәеӨұжғ…еҶөпјҢе№¶еңЁе…ій”®жЁЎжҖҒдҝЎжҒҜзјәеӨұж—¶д»ҚиғҪдҝқжҢҒй«ҳеҲҶеүІзІҫеәҰзҡ„йІҒжЈ’з®—жі•гҖӮ
